Our mission is to visualize the US Presidential Election results from 2000 â€“ 2016 through an interactive dashboard. The user can navigate the nation map by using a dropdown menu to display the election results numerically for each state by election year, and the map adapts a color-coded choropleth scheme proportionate to the vote counts for each party. In addition, the user can click on a state to further analyze election results by county. Finally, a line chart summarizes the total vote counts for each party by election year.

*The data in our interactive dashboard comes from Harvard's Dataverse Repository and the Exploratory GeoJson maps, files & data site.
